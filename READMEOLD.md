<a  name="node"></a>Install Node dependencies

```
$ yarn install
$ npm install

```
## <a name="scripts"></a>Scripts
Webpack watch:
```
$ yarn watch
$ npm run watch
```

Webpack build (minified):

```

$ yarn build
$ npm run build

```
Linting with eslint:

```
$ yarn lint
$ npm run lint
```
## Gulp tasks

Compile LESS:
```
$ gulp compile
```
Compile and watch LESS:
```
$ gulp watch
```
CLDR locale data for internationalization:
```
$ gulp cldr
```
